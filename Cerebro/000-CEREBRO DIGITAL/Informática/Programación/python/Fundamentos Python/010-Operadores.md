Los _**operadores**_ disponibles son:
```
Relacionales:
- >= , <=, <, > : Mayor o igual, menor o igual, mayor o menor
- != , == : Distinto, Igual
- in : Contenido por 
Lógicos:
- not o ~ : Negación
- and o & : Ambas verdaderas
- or o | : Una u otra es verdadera
```
Como booleanos, se pueden utilizar tanto 0 y 1 como True y False.
